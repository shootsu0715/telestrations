# 🎨 テレストレーション - お絵描き伝言ゲーム

社内懇親用のマルチプレイヤーお絵描き伝言ゲームです。
同じWiFiに接続した複数のPCやスマホからアクセスして遊べます。

## 🚀 セットアップ

### 必要なもの
- Node.js (v16以上)

### インストール & 起動

```bash
cd telestrations
npm install
npm start
```

起動すると以下のように表示されます：

```
🎨 ==========================================
   テレストレーション サーバー起動！
==========================================

   🏠 ローカル:  http://localhost:3000
   📱 ネットワーク: http://192.168.x.x:3000

   ↑ 同じWiFiの端末からアクセスできます
==========================================
```

### アクセス方法
- ホスト（サーバーを起動したPC）: `http://localhost:3000`
- 他の端末: `http://192.168.x.x:3000`（表示されたネットワークアドレス）

## 🎮 遊び方

1. **ルーム作成**: ホストが名前を入力して「ルームを作る」をクリック
2. **参加**: 他のプレイヤーはルームコード（4文字）を入力して参加
3. **ゲーム開始**: 3人以上集まったらホストが「ゲーム開始」
4. **お題入力**: ホストが自由にお題を入力
5. **描く**: 最初のプレイヤーがお題を見て絵を描く（制限時間あり）
6. **当てる**: 次のプレイヤーが絵を見て答えを入力
7. **繰り返し**: 描く→当てる→描く→当てる... を全員分
8. **結果発表**: 紙芝居形式で1枚ずつめくって結果を楽しむ！

## ⚙️ 設定

- **制限時間**: ロビーで30秒〜120秒に変更可能（ホストのみ）
- **プレイヤー数**: 3〜8人
